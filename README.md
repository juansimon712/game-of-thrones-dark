# Game Of Thrones Dark

A dark theme for Omarchy with red and gold accents on black, inspired by the Iron Throne.

## Preview

- **Background:** Black with weathered gold metallic text
- **Accent color:** `#CC0000` (Targaryen red)
- **Foreground:** `#FFD700` (gold)
- **Background:** `#000000` (pure black)

## Wallpapers

Includes Iron Throne and sigil-themed wallpapers.

## Font

Requires the **Game of Thrones** font by Charlie Samways (included - free for personal use).
